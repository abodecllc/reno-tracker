First time setup

This tracker syncs between family members using a free cloud storage bin. You only need to set this up once — then share the key with your family.

Step 1 — Get a free API key

Go to jsonbin.io, create a free account, then copy your Master Key from the API Keys section.

Join existing tracker

Enter the tracker key shared by your family member, your JSONBin Master Key, and your name.

Tracker created!

Share the key below with your family. Each person needs their own free JSONBin account and Master Key — they'll use "Join existing tracker" and paste in this key.
